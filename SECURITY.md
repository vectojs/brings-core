# Security Policy

## Supported versions

Security fixes are applied to the latest published `@vectojs/brings-core`
release.

## Reporting a vulnerability

Please report potential vulnerabilities privately through the
[GitHub Security Advisories](https://github.com/vectojs/brings-core/security/advisories/new)
flow. Do not open a public issue before maintainers have had a chance to assess
and coordinate a fix.
